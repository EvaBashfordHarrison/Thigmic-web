## About 
Thigmic is a language system to prove non-human intelligence in plant species. Designed to be viewed with the Thigmic Reader; an ECG module picking up plant signals. 

## Web app 
The web app is designed for users to interact with the language system, with a single glyph viewer, and a 'typing' mode.
